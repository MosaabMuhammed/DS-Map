Scatter plot with Log axis¶

def log_trans(x, inverse = False):
    if not inverse:
        return np.log10(x)
    else:
        return np.power(10, x)

sns.regplot(df['displ'], df['comb'].apply(log_trans))
tick_locs = [10, 20, 50, 100, 200, 500]
plt.yticks(log_trans(tick_locs), tick_locs);

Overplotting, Transparency, and Jitter¶

  • Since year is a discrete variable, we need to use Transparency and jitter to see the true relationship.
In [17]:
plt.scatter(data=df, x='year', y='comb');
In [19]:
plt.scatter(data=df, x='year', y='comb', alpha=1/20);
  • As you can see above, there are some dense points in the bottom area of the figure, now we need to see more on this relationship by using jitter.
  • But Matplotlib doesn't have built-in parameter for jitter, therefore, we're going to use seaborn.
In [29]:
sns.regplot(data=df, x='year', y='comb', x_jitter=.4, y_jitter=.2,
            scatter_kws = {'alpha': 1/20}, fit_reg=False);
