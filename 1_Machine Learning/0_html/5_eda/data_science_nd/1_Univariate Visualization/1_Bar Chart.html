Order Nomial Variable¶

Since this variable is Nomial, we should order it from the most common one to the least.

In [6]:
base_color = sns.color_palette()[1]
embarked_order = titanic['Embarked'].value_counts().index
sns.countplot(data=titanic, x='Embarked', color=base_color,
              order=embarked_order);

Don't Order Ordinal variable¶

  • Pclass is an ordinal varaible since it has a ranking values
    • class 1 for high level.
    • class 2 for medium level.
    • class 3 for low level.
In [7]:
base_color = sns.color_palette()[1]
pclass_order = [1, 2, 3]
sns.countplot(data=titanic, x='Pclass', color=base_color, 
             order=pclass_order);

From Absolute to Relative Values¶

Instead of showing the real counts of the labels, we need to show it as the value of all the labels range from 0 to 1, therefore we can show some percentage or whatever we need.

In [12]:
# Get proportion taked by most common group for derivation of tick marks
n_points = titanic.shape[0]
max_count = titanic['Pclass'].value_counts().max()
max_prop = max_count / n_points
In [18]:
# Generate tick mark locations and names
tick_props = np.arange(0, max_prop, .05)
tick_names = ['{:.2f}'.format(label) for label in tick_props]
In [19]:
# Create the plot 
base_color = sns.color_palette()[1]
sns.countplot(data=titanic, x='Pclass', color=base_color);
plt.yticks(tick_props * n_points, tick_names)
plt.ylabel('Proportion');

Show Relative Value on Each Bar¶

  • We can use text annotation to label the frequencies on bars instead.
  • This requires writing a loop over the tick locations and labels and adding one text element for each bar.
In [32]:
# Create the plot
base_color = sns.color_palette()[1]
sns.countplot(data=titanic, x='Embarked', color=base_color);

# Add Annotations
n_points = titanic.shape[0]
embarked_count = titanic['Embarked'].value_counts()
locs, labels = plt.xticks() # Get the current tick locations and labels

# Loop through each pair of locations and labels
for loc, label in zip(locs, labels):
    # Get the text property for the lable to get the correct count.
    count = embarked_count[label.get_text()]
    pct_string = '{:.01f}%'.format(100*count/n_points)
    
    # Print the annotation just below the top of the bar.
    plt.text(loc, count-40, pct_string, ha='center', color='w')
