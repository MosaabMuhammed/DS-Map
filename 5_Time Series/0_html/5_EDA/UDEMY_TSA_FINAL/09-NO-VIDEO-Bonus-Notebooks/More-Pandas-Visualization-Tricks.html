Quickly add one axis label¶

We mentioned that although we can pass a title into the pandas .plot() function, we can't pass axis labels this way.
However, we can quickly add a label to either the x- or y-axis by running a .set_xlabel() or .set_ylabel() method:

In [2]:
df2.plot(figsize=(8,3), title='Custom Pandas Plot').set_xlabel('X-axis data');

Unfortunately, we can't add both labels this way:

In [3]:
df2.plot(figsize=(8,3)).set_xlabel('X-axis data').set_ylabel('Y-axis data');

Remove unwanted axis labels¶

Sometimes pandas automatically places the column name as an x or y axis label. To remove them we can call the label on the axis object and set_visible to False. Alternatively, we can simply pass in empty strings as labels.

In [4]:
df2.plot.scatter(x='a',y='b',figsize=(4,2));
In [5]:
ax = df2.plot.scatter(x='a',y='b',figsize=(4,2))
ax.xaxis.get_label().set_visible(False)
ax.yaxis.get_label().set_visible(False)
In [6]:
# ALTERNATIVELY:
ax = df2.plot.scatter(x='a',y='b',figsize=(4,2))
ax.set(xlabel='', ylabel='');

Adding commas to y-axis values¶

In [11]:
import matplotlib.ticker as ticker
formatter = ticker.StrMethodFormatter('{x:,.0f}')

df=pd.DataFrame([1000000,800000,60000,200,2000],index=range(5))

ax = df.plot(figsize=(4,2),legend=False)
ax.autoscale(axis='x',tight=True)
ax.set(xlabel='', ylabel='')
ax.yaxis.set_major_formatter(formatter);
